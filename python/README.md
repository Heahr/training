### 1. 코딩블럭 들여쓰기(Identation)

Python은 코딩블럭을 구분하기 위해 들여쓰기를 사용한다. 기존의 C/C++, JAVA 등의 언어에서는 세미콜론, Curly Bracket({...})을 사용했지만 Python은 이를 '들여쓰기(Identation)로 구분하는 것이다.  

일반적으로 들여쓰기는 4개의 공백을 사용할 것을 권장한다. 이와 함께 동일한 블럭의 코드는 모두 동일한 수의 공백을 사용해야 한다. 즉, 4개의 공백을 사용하는 코드블럭에서 5개의 블럭을 사용하는 코드블럭을 추가한다면 에러가 생기는 것이다.

> 또한, 공백(Space)과 Tab을 구분해야 한다. 이는 쉽게 저지를 수 있는 오류이며, 개발 세계에는 스페이스파와 탭파로 구분하여 양분되어 있기 때문에 무엇이 옳다고 할 수 없다.  
>
> 여기에 대한 책으로는 '임백준'님의 '3 Space Indentation'이 있다.

##### 사용 예

```
x = 1

if x > 0:
	a = 1
	b = 2
	c = a + b
else:
	a = -1
	b = -2
	c = a - b
```

### 2. 표준 라이브러리

파이썬은 상당히 많은 표준 라이브러리를 제공한다. 이 표준 라이브러리를 호출해 사용하기 위해서는 `import`문을 사용해야 한다. 다음과 같이 말이다.

##### 사용 예

```
import math

n = math.sqrt(9.0)

print(n)
```

물론 `import`문을 사용하는 방법에는 여러가지가 있지만, 여기서는 가장 가본적인 형태만 설명하도록 하겠다. 위의 문장 중 

##### 사용 예

```
n = math.sqrt(9.0)
```

위 문장은 math 라이브러리 속의 sqrt()라는 함수에 매개변수 `9.0`을 대입해 `n`이라는 변수에 저장하라는 문장이다. 이를 통해 라이브러리를 호출하는 방법에 대해 이해했다.

### 3. 주석

Pyton에서는 주석을 나타내기 위해 `#`을 사용한다. 일반적으로 라인이 시작되기 전 혹은 라인의 끝에 온다. 또한, 표준 코딩 스타일에서는 # 다음에 한 칸을 띄어쓰는 것을 권장한다.

### 4. PEP(Python Enhancement Proposals)

파이썬의 공식 문서를 말한다. 여기에는 파이썬을 개선하기 위한 제안들이 적혀있다. 다음과 같은 주제로 말이다.

	1. 새로운 기능의 추가, 구현 방식 제안(Standard Track PEP)
	2. 디자인 이슈 설명, 일반적인 가이드라인 혹은 정보(Informational PEP)
	3. Python을 둘러 싼 프로세스에 대한 설명, 프로세스 개선 제안(Process PEP)

PEP는 파이썬 언어 자체만이 아니라 코딩 표준 스타일, 커뮤니티 이슈 등을 담고 있는 유용한 자료이다.

*  [https://www.python.org/dev/peps/](https://www.python.org/dev/peps/)

### 5. Python에 날개달기

일반적인 파이썬은 C/C++보다 느리다는 인식이 강하다. 왜냐하면 속도를 중시하기 보다는 간결한 문장, 가독성 등이 고려된 언어이기 때문이다. 그런데 이러한 파이썬이 왜 속도가 중요한 인공지능, 공학 등에서 사용할 수 있는 걸까?

거기에 대한 답은 '추가 라이브러리'에서 찾을 수 있다. 가령 'Numpy'나 'Scipy'와 같은 라이브러리가 Python의 속도를 향상 시켜 C/C++보다 빠르게 연산을 처리할 수 있도록 도와주는 것이다.

> 그래도 Python에 날개를 달기 위해서는 걷는 법부터 알아야 한다. 이를 위해서 Python에 익숙해지도록 하자.

### 1. 파이썬의 기본 데이터 타입

|타입|설명|표현 예|
|:--:|:--:|:-----:|
|int|정수|a = 100, a = 0xff(16진수), 0o56(8진수)|
|flout|소숫점을 포함한 실수|a = 10.25|
|bool|참, 거짓을 표현|a = True|
|None|Null과 같은 표현|a = None|

파이썬의 표현 방식은 C/C++, JAVA 등과는 다르다. 파이썬에서는 자료형을 표현하지 않기 때문에 편리하게 변수를 사용할 수 있다.

> 그러나 변수를 변환하는 작업을 컴퓨터가 하기 때문에 앞서 언급한 C/C++에 비해 속도가 느릴 수 밖에 없다. 이를 해결하기 위해서는 추가적인 라이브러리의 사용이 필수적이다.

##### 사용 예

```
int(3.5)      # 3
2e3           # 2000.0
float("1.6")  # 1.6
float("inf")  # 무한대
float("-inf") # -무한대
bool(0)       # False. 숫자에서 0만 False임,
bool(-1)      # True
bool("False") # True
a = None      # a는 None
a is None     # a가 None 이므로 True
```

### 2. 연산자

파이썬은 산술, 비교, 할당, 논리, Bitwise, 멤버쉽, Identity 연산자를 지원한다.

#### 산술 연산자

사칙 연산자(+, -, \*, /)와 제곱 연산자(\*\*), 나머지 연산자(%), 그리고 나누기에 소숫점 이하를 버리는 연산자(//) 등이 있다.

##### 사용 예

```
5 % 2   # 1
5 // 2  # 2
```

#### 비교 연산자

등호(==), 같지 않음(!=), 부등호(<, >, <=, >=) 등이 있다.

##### 사용 예

```
if a != 1:
   print("1이 아님")
```

#### 할당 연산자

변수에 값을 할당하기 위해 사용하는 연산자(=)로, 산술 연산자와 함께 사용되어 간결한 표현을 나타낼 수도 있다(+=, -=, \*=, /=, %=, //=).

##### 사용 예

```
a = a * 10
a *= 10     # 위와 동일한 표현
```

#### 논리 연산자

파이썬에서는 and, or, not과 같은 단어를 통해 논리 연산을 진행한다.

```
x = True
y = False
 
if x and y:
    print("Yes")
else:
    print("No")
# No가 출력된다.
```
